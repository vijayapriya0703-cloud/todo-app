# 📝 To-Do List App

A fully functional To-Do List web application 
built with pure HTML, CSS, and JavaScript.

## 🌐 Live Features
- ✅ Add, edit and delete tasks
- ⭐ Priority levels (High, Medium, Low)
- 🗂️ Categories (Work, Personal, Shopping)
- 📅 Due dates for each task
- 🔍 Search tasks instantly
- 🌙 Dark mode / Light mode toggle
- 💾 Tasks auto-saved with localStorage
- 🔢 Task counter
- 🗑️ Clear completed tasks

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## 💡 What I Learned
- DOM manipulation with JavaScript
- localStorage for saving data in browser
- CSS variables for dark/light theme switching
- Responsive and clean UI design

## 🚀 How to Run
1. Download or clone this repository
2. Open index.html in any browser
3. Start adding your tasks!
